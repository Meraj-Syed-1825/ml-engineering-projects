# ML Engineering Projects

A structured self-study journey through Machine Learning, built on an engineering 
background in metallic materials and metal forming.

All projects use a **steel materials dataset** — predicting yield strength from 
composition and process parameters (Carbon %, Manganese %, Rolling Temperature).

## Engineering Context

The steel dataset simulates real materials data:
- **Features:** Carbon %, Manganese %, Rolling Temperature (°C), Silicon %, Chromium %
- **Target:** Yield Strength (MPa)
- **Domain insight:** Manganese solid solution strengthening — high strength achievable 
  at low carbon, relevant for weldability in structural steels
